# BLUMII COSMETICS - E-commerce Website

Stack: **NodeJS + Express + ReactJS + MongoDB**

BLUMMI COSMETIC is an online cosmetics sales website, supporting shopping, account registration, cart management, orders, delivery addresses, and admin management system.
